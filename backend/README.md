# Backend

Este es el directorio del backend del proyecto.

## Estructura

La estructura de este directorio debe seguir las mejores prácticas del framework/tecnología que uses (Node.js, Python, etc.).

## Instalación

```bash
# Instalar dependencias
pnpm install
```

## Scripts Disponibles

```bash
# Desarrollo
pnpm dev

# Build
pnpm build

# Linting
pnpm lint

# Tests
pnpm test
```

## Configuración

Crea un archivo `.env` basado en `.env.example` (si existe) y configura las variables de entorno necesarias.

Para más información sobre la estructura del proyecto, consulta la [documentación principal](../docs/).
