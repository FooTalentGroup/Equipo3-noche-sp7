# Frontend

Este es el directorio del frontend del proyecto.

## Estructura

La estructura de este directorio debe seguir las mejores prácticas del framework/tecnología que uses (React, Vue, Next.js, etc.).

## Instalación

```bash
# Instalar dependencias
pnpm install
```

## Scripts Disponibles

```bash
# Desarrollo
pnpm dev

# Build
pnpm build

# Linting
pnpm lint

# Tests
pnpm test
```

## Configuración

Crea un archivo `.env` basado en `.env.example` (si existe) y configura las variables de entorno necesarias.

## Deploy

Configura el deploy según tu plataforma (Vercel, Netlify, etc.). Ver [documentación de Vercel](../docs/VERCEL_CONFIG.md) para más detalles.

## Documentación

Para más información sobre la estructura del proyecto, consulta la [documentación principal](../docs/).
